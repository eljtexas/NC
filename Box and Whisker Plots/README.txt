This folder contains scripts for generating boxplots for the
healthy donor samples and the VRC26 longitudinal timepoints

Scripts:

1. make_figure1_boxplots_healthy_donor_points_density.py
 - generates boxplots for naive/memory samples
 - input files must be generated using scripts from 
"HD1 Heavy Chain Memory-Naive"

2. V_gene_intron_divergence_plots_mafft_non_multiplex_IGHJ3.py
 - generates 2D histogram divergence plots for a given VRC26 heavy chain time point
 - generates input files for make_figure1_boxplots_VRC26_density.py

3. V_gene_intron_divergence_plots_mafft_non_multiplex_IGLJ1.py
 - generates 2D histogram divergence plots for a given VRC26 light chain time point
 - generates input files for make_figure1_boxplots_VRC26_density.py

4. make_figure1_boxplots_VRC26_density
 - generates boxplots for VRC26 longitudinal samples
